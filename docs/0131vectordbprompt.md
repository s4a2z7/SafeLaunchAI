## 참고사항

- **임베딩 방식:** Python 3.14 환경에서 ChromaDB 미지원으로 TF-IDF (char_wb, ngram 1-3) 사용
- **score_threshold:** 기본 0.7 (카더라 방지), 테스트 시 0.05로 완화하여 검증
- **store_policies 컬렉션:** 현재 0건 (스토어 정책 데이터는 향후 별도 수집 필요)
- **데이터 갱신:** `sync_legal_data()` 함수로 주기적 동기화 가능
