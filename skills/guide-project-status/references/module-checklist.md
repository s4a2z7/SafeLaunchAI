```python
    "text_samples": ["문구1", "문구2"],
    "payment_related": ["결제 관련 텍스트"],
    "privacy_related": ["개인정보 관련 텍스트"]
}
```

### 3. core/legal_rag.py (ChromaDB 검색)

**담당:** 팀원 A

**구현 체크리스트:**
- [ ] ChromaDB 연결 설정
- [ ] 쿼리 임베딩 생성
- [ ] 유사도 검색 실행
- [ ] 유사도 임계치 필터링 (카더라 방지)
- [ ] 문서 리스트 반환

**입력:** 쿼리 문자열, 옵션(상위 k개, 점수 임계치)
**출력:** 문서 리스트 (텍스트, 메타데이터, 유사도 점수)

```python
# 예상 출력 형식
[
    {
        "text": "법령 내용",
        "metadata": {"source": "저작권법", "article": "제30조"},
        "score": 0.85
    }
]
```
